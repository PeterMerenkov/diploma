start
:Открыть файл нечетких правил;
:Создать пустой список для хранения нечетких правил;
repeat :Итерация по строкам файла;
  :Пропустить первые ячейки, не содержащие данных нечетких правил;
  :Создать списки для индексов параметров и терм-множеств;
  repeat :Итерация по ячейкам текущей строки;
    :Добавить индекс параметра в соответствующий список;
  repeat while (Есть еще ячейки в строке)
  :Перейти к следующей строке;
  repeat :Итерация по ячейкам текущей строки;
    :Добавить индекс терм-множества в соответствующий список;
  repeat while (Есть еще ячейки в строке)
  :Создать список пар индексов параметров и терм-множеств;
  :Перейти к следующей строке;
  :Прочитать результат из текущей строки;
  :Добавить новое нечеткое правило в список;
repeat while (Есть еще строки в файле)
:Вернуть результат;
stop
